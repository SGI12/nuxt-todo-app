# Notes App

Todo list nuxt js

## Quickstart

### Docker

```bash
docker-compose up
```

Open http://localhost:3000

### Local dev

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack

- Nuxt.js 3
- Vue 3 (Composition API)
- TypeScript
- Pinia (state management)
- localStorage
- Docker

## Commands

```bash
# Development
npm run dev

# Build
npm run build

# Preview production build
npm run preview

# Docker
docker-compose up
docker-compose down
```
